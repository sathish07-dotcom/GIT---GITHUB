Welcome to Git & GitHub Guide

Learn the fundamentals of Git and GitHub, version control, and important commands.

What is Git?

Git is a distributed version control system that helps developers track changes in their codebase, collaborate with others, and manage different versions of their projects.

Features of Git:

  • Tracks changes in the code.
  • Allows multiple developers to collaborate.
  • Provides branching and merging features.
  • Stores project history securely.

What is GitHub?

GitHub is a cloud-based platform that allows developers to store, share, and collaborate on Git repositories.

Features of GitHub:

  • Host Git repositories online.
  • Enable collaboration with pull requests.
  • Provide issue tracking and project management tools.
  • Support CI/CD workflows and integrations.

Basic Git Commands

Command Description
git init Initialize a new Git repository.
git clone <repo_url> Clone an existing repository.
git status Check the status of your repository.
git add . Add all files to staging area.
git commit -m "message" Commit changes with a message.
git push origin main Push changes to the remote repository.
git pull origin main Pull latest changes from the remote repository.
git branch List all branches.
git checkout -b new-branch Create and switch to a new branch.
git merge branch-name Merge a branch into the current branch.
